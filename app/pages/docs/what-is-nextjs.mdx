---

Next.js está integrado en Blitz y casi todas las funciones de Next.js están soportadas. Para obtener más información sobre las diferencias, consulta aquí: [¿Por qué usar Blitz en lugar de Next.js?](./why-blitz)

[**Next.js**](https://nextjs.org/) es un popular _framework_ de React con muchas funciones integradas, tales como:

* _Pre-rendering_, tanto la generación estática \(SSG\) como el renderizado del lado del servidor \(SSR\) están soportadas por cada página individualmente
* División automática de código para cargas de página más rápidas
* Enrutamiento del lado del cliente con _pre-fetching_ optimizado
* Entorno de desarrollo que con soporte para _Hot Module Replacement_
* Totalmente extensible
